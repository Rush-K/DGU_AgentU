기업사회 프로젝트 (위챗 미니프로그램)
----------------------------

## 동국대학교 중국인 유학생을 위한 소액 결제 미니프로그램
### 주요 기능
	- ##### 교내 편의시설 위치 및 운영시간 정보
	- ##### 학생 식당 식품 알레르기 정보
	- ##### WeChat 간편결제(IMFORM API) 기능
	- ##### 파파고 번역 API를 통한 중-한, 한-중 텍스트 번역
	- ##### 결제내역 확인, 식권 발급 및 사용 기능
